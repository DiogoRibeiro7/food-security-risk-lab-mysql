# Responsible use

Food-security analysis is high stakes. This repository must be used carefully.

## What the project can do

- Highlight deterioration in selected indicators.
- Create reproducible risk scores.
- Help analysts prioritize further review.
- Compare countries and years under a transparent scoring framework.

## What the project cannot do

- Declare famine.
- Prove causality.
- Replace field assessment.
- Capture conflict, access constraints, household coping strategies, or local market failures unless such data are added explicitly.

## Recommended wording

Use:

- "risk signal"
- "early-warning indicator"
- "requires expert review"
- "deterioration relative to baseline"

Avoid:

- "famine prediction"
- "food insecurity confirmed"
- "caused by rainfall"
- "humanitarian emergency detected"
